# Article Web Page

A static article web page with slideshow and multiple content sections including welcome, articles, space, and about pages.

## Tech Stack

- HTML5
- CSS3 (with slideshow animations)

## Features

- Landing / Welcome page
- Multiple article sections (main, space, articles listing)
- About page
- Image slideshow with CSS animations
- Clean, responsive layout
